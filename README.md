# Rayo Washing

Sitio web estático para **Rayo Washing**, una lavandería autoservicio en Abrera.

La web presenta el negocio, sus servicios, precios, fotografías reales del local, ubicación, contacto por teléfono/WhatsApp y páginas legales básicas.

## Demo

[https://lavanderia-miguel.vercel.app/](https://lavanderia-miguel.vercel.app/)

## Tecnologías

- HTML
- CSS
- JavaScript
- Vercel

## Estructura

```text
.
+-- index.html
+-- aviso-legal.html
+-- privacidad.html
+-- styles.css
+-- script.js
+-- assets/
    +-- icons/
    +-- real/
```

## Páginas

- `index.html`: página principal.
- `aviso-legal.html`: aviso legal del negocio.
- `privacidad.html`: información básica de privacidad.

## Funcionalidades

- Navegación responsive.
- Menú móvil.
- Secciones de servicios, precios, mascotas, funcionamiento, ubicación y contacto.
- Galería con fotos reales del local.
- Mapa embebido de Google Maps.
- Formulario visual de contacto con mensaje informativo.
- Enlaces directos a teléfono, WhatsApp y Google Maps.
